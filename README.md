# Colman College - Computer Science B.Sc

This repository contains the Lumina Study course structure for the Computer Science B.Sc program at the College of Management Academic Studies (Colman).

## Program Overview

The degree combines foundational computer science courses with specialized tracks in Data Science and AI. Students can choose their specialization after the first year.

## Structure

- **Year 1**: Foundation courses in mathematics and programming
- **Year 2**: Core computer science courses
- **Year 3**: Specialization tracks and capstone project

## Specialization Tracks

- Data Science
- Cybersecurity
- Deep Learning
- Full Stack Development
- Fintech
